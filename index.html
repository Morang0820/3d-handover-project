01 — Abstract

Abstract

We propose a low-stimulation asynchronous handover framework for 3D product designers collaborating across time zones. Existing asynchronous handoff tools rely on 2D logs or passive video recordings, which fail to convey spatial relationships and design intent. Replacing these with full 3D recordings reviewed inside an HMD introduces cybersickness and cognitive overload from rapid immersive scrubbing.

Our system addresses both problems simultaneously. An LED Wall provides a low-fatigue 2D overview for the majority of review time, while HMD interaction is reserved for short, purposeful, on-demand 3D inspection. Designers navigate a spatial design history timeline using a mid-air dial gesture with visual and haptic cues, then pull moments into 3D space for closer inspection before returning to the 2D flow.

02 — System

Interaction Design

The interaction follows a four-step flow that keeps immersive use short and purposeful, reducing cybersickness while preserving the spatial richness of 3D design history.

STEP 01

Browse timeline

Scroll through keyframes on the LED Wall using a mid-air dial gesture (wrist rotation).

STEP 02

Receive cues

Visual flash and haptic pulse fire simultaneously at each keyframe, drawing attention without conscious effort.

STEP 03

Inspect in 3D

Pull a moment into 3D space via a pull-to-spawn gesture for closer spatial inspection.

STEP 04

Return to 2D

Push-to-dismiss sends the object back; resume 2D timeline scrubbing.

Multimodal Cue Design

Four cue events are defined with uniform intensity across all keyframes. Channels are separated: visual handles continuous state, haptic handles immediate event alerts, and audio is reserved for out-of-FoV events only.

03 — Prototype

Prototype

The prototype implements the full interaction pipeline on a physical LED Wall with a Meta Quest 3 HMD. Three input methods are compared: pinch gesture, controller, and the proposed mid-air dial (wrist rotation). All three control the same 3D timeline bar. Haptic feedback is delivered via bHaptics Tactgloves (Unity 6000.3.4f1).

04 — Evaluation

Evaluation Plan

The experiment adopts a 3×2 within-subjects factorial design, crossing three input methods with two cue conditions (6 conditions total). Participants: N = 8 adults aged 20–35 (pilot study); 5 trials per condition, 30 trials total.

Metrics

TKAT

Target Keyframe Acquisition Time

Time (ms) from trial start to successful arrival at the target keyframe. Adapted from Fitts [1].

Overshoot

Overshoot Rate

Frequency of passing a keyframe without stopping (count per trial).

WRS

Workflow Reconstruction Score

Card sorting + comprehension questions testing accuracy of sequence reconstruction. Adapted from Gou et al. [3].

NASA-TLX

Cognitive Load

Subjective load (0–100) across 6 subscales. Reference delta: ~14 pts between AR+haptic vs. AR-only [11].
